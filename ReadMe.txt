1-Solution is written in TestNG Framework.
2-After running the test suite an Extent report is generated with details for each test case/test suite and it is also included in the project.
3-After running the test suite please refresh the "Screenshots" folder to get the latest screen shots which is considered a part of the validations along with the test cases assertions.
4-The project consists of 3 packages where we can describe them with the following details
	a. The first one is related to the portal pages that includes all the locators and the methods concerned with that page.
	b. The second one is related to the test cases with needed assertions and validations.
	c. The third one is related to the utilities where it includes different methods needed in the execution of test cases.
5-The project was tested using Chrome driver and it supports other drivers.
